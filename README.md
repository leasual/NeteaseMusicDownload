# NeteaseMusicDownload
网易云音乐批量下载器，支持:

下载音乐/歌词/封面

支持多种比特率

自定义命名

vip音乐下载（vip用户only）

---

使用 python3.8

# api

from：

https://github.com/Binaryify/NeteaseCloudMusicApi

# requirement

requests

eyed3（可选）

how to install：

```
pip3 install requests
pip3 install eyed3
```
